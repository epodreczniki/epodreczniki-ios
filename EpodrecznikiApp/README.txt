READEME

1. On OS X system open App Store and download Xcode
2. Double click on EpodrecznikiApp.xcodeproj
3. Open file EPConstants.h and provide URL to API in line no. 2 (API_BASE)
4. From menu Xcode choose Preferences then Accounts and add your developer account
5. In Project navigator choose EpodrecznikiApp then Target EpodrecznikiApp then General Tab. Inside Identity group update your app bundle identifier (default value is pl.edu.ore.EpodrecznikiLite)
6. From menu Product choose Run
